# Worklog Studio

A lightweight desktop app for tracking work time and tasks.

Minimal interface, fast log input, and focus on staying in the flow without distractions.

## ✨ Features

- time tracking
- simple work logs
- macOS app built with Flutter
- auto-updates via Sparkle

## 🚀 Tech Stack

- Flutter (macOS)
- Firebase (auth / backend)
- Sparkle (auto-updates)

## 🛠 Development

```bash
flutter run -d macos
```
